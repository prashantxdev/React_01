## 🛠️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/prashantxdev/React_01.git
```

2. Go to project folder:

```
cd 01-folder
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm run dev
```

5. Open in browser:

```
http://localhost:5173/
```
